城市：苏州

适合人群：
江南园林、古城漫步、历史文化、亲子游、情侣旅行、摄影、古镇体验、美食探索、周末短途旅行

推荐游玩天数：
2-4 天

最佳旅行季节：
苏州最适合旅游的季节是春季 3-5 月和秋季 9-11 月。

春季气候温和，园林花木繁盛，适合游览拙政园、留园、狮子林、平江路和山塘街。秋季天气凉爽，古城街巷和园林景色更有江南韵味，适合城市漫步、拍照和品尝苏帮菜。

夏季较热且湿度较高，建议把园林和古街安排在上午或傍晚。冬季游客相对少，古城氛围安静，适合慢节奏游览和体验评弹、茶馆、博物馆。

城市概览：
苏州是中国最具代表性的江南城市之一，以古典园林、运河古城、评弹昆曲、苏帮菜和水乡古镇闻名。这里既有拙政园、留园、狮子林、苏州博物馆等经典文化景点，也有平江路、山塘街、金鸡湖、同里古镇、周庄古镇等适合休闲游览的区域。

苏州适合喜欢江南文化、园林建筑、古街漫步、摄影和慢旅行的游客。第一次来苏州建议安排 2-3 天；如果想去同里、周庄、甪直等古镇，可以安排 4 天左右。

交通信息：
苏州没有大型民航机场，游客通常从上海虹桥、上海浦东、无锡硕放机场或杭州萧山机场中转前往。高铁是前往苏州最方便的方式之一，苏州站靠近古城区，苏州北站主要服务高铁线路，距离市区稍远。

市内交通以地铁、公交、出租车和网约车为主。苏州古城区景点较集中，平江路、拙政园、苏州博物馆、狮子林等可以步行或短途打车串联。金鸡湖位于苏州工业园区，适合乘坐地铁或打车前往。

交通建议：
- 古城区道路较窄，节假日容易拥堵，建议优先选择地铁和步行。
- 拙政园、狮子林、苏州博物馆和平江路可以安排在同一天。
- 山塘街适合傍晚和晚上游览，夜景更有氛围。
- 去同里、周庄等古镇建议预留半天到一天。
- 雨天的苏州很有江南意境，但古街石板路湿滑，建议穿防滑鞋。

住宿区域推荐：
1. 观前街 / 平江路附近
适合第一次来苏州的游客。这里靠近古城区核心景点，餐饮、购物和交通都方便。

2. 山塘街附近
适合喜欢古街夜景和江南水巷氛围的游客。晚上游览方便，但热门时段游客较多。

3. 苏州站附近
适合高铁到达、短途停留或行程紧凑的游客，交通便利。

4. 金鸡湖 / 苏州中心附近
适合商务出行、现代城市体验和喜欢湖景夜景的游客。住宿条件较新，但离古城区稍远。

5. 同里 / 周庄古镇周边
适合想住一晚水乡古镇、体验清晨和夜晚安静氛围的游客。

热门景点：
1. 拙政园
类型：古典园林 / 世界遗产 / 江南文化
推荐游玩时间：2-3 小时
亮点：江南园林代表、亭台楼阁、水景、花木
建议：游客较多，建议早上前往。可与苏州博物馆、狮子林安排在同一天。

2. 苏州博物馆
类型：博物馆 / 建筑 / 文化
推荐游玩时间：1.5-2 小时
亮点：贝聿铭建筑、苏州文物、江南美学
建议：通常需要提前预约，适合文化和建筑爱好者。

3. 狮子林
类型：古典园林 / 假山 / 文化
推荐游玩时间：1-1.5 小时
亮点：假山迷宫、园林建筑、江南庭院
建议：适合亲子游和第一次来苏州的游客。

4. 留园
类型：古典园林 / 世界遗产
推荐游玩时间：1.5-2 小时
亮点：园林空间、太湖石、古建筑
建议：比拙政园相对安静，适合慢慢欣赏园林细节。

5. 平江路
类型：古街 / 城市漫步 / 美食
推荐游玩时间：1-2 小时
亮点：水巷、老宅、咖啡馆、苏式小吃
建议：适合下午或傍晚慢走，商业化较强但氛围好。

6. 山塘街
类型：古街 / 夜景 / 运河
推荐游玩时间：1-2 小时
亮点：七里山塘、运河夜景、江南水巷、小吃
建议：晚上灯光亮起后体验更好，节假日人多。

7. 寒山寺
类型：寺庙 / 文化 / 诗词
推荐游玩时间：1-1.5 小时
亮点：枫桥夜泊、古寺、钟声文化
建议：适合喜欢诗词文化和寺庙氛围的游客。

8. 金鸡湖
类型：湖景 / 现代城市 / 夜景
推荐游玩时间：1-2 小时
亮点：湖景、苏州中心、东方之门、夜景
建议：适合傍晚和晚上游览，可安排湖边散步。

美食推荐：
1. 松鼠桂鱼
苏帮菜代表，外脆里嫩，酸甜口味明显。
适合：本地菜体验、多人用餐

2. 苏式面
如焖肉面、爆鱼面、虾仁面等，汤头清鲜，适合早餐或午餐。
适合：本地早餐、快速简餐

3. 生煎 / 小笼
苏州常见小吃，适合作为早餐或点心。
适合：小吃爱好者

4. 蟹粉菜品
秋冬季尤其受欢迎，如蟹粉面、蟹粉豆腐等。
适合：季节美食爱好者

5. 桂花糖藕
江南甜品，软糯香甜，带有桂花香气。
适合：甜品爱好者

一日游路线：
主题：苏州古城经典路线
上午：拙政园、苏州博物馆
午餐：平江路或观前街品尝苏式面、苏帮菜
下午：狮子林、平江路城市漫步
晚上：山塘街夜景和小吃

二日游路线：
第 1 天：拙政园、苏州博物馆、狮子林、平江路、山塘街
第 2 天：留园、寒山寺、虎丘，晚上金鸡湖夜景

三日游路线：
第 1 天：苏州古城区经典景点
第 2 天：留园、虎丘、寒山寺、金鸡湖
第 3 天：同里古镇或周庄古镇一日游

购物与伴手礼：
苏式糕点、碧螺春茶、苏绣、桃花坞木版年画、采芝斋糖果、苏式月饼、园林文创产品

旅行贴士：
- 苏州园林建议提前预约或提前购票。
- 古城区适合步行，但一天不要安排太多园林，容易审美疲劳。
- 山塘街和平江路节假日人很多，建议错峰游览。
- 苏州适合慢旅行，建议多留时间在古街、茶馆和园林里。
- 春秋季是苏州旅游旺季，住宿建议提前预订。
