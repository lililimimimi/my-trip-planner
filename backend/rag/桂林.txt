城市：桂林

适合人群：
山水风光、漓江游船、竹筏漂流、阳朔旅行、亲子游、摄影、自然风光、背包旅行、喀斯特地貌、田园骑行

推荐游玩天数：
2-4 天

最佳旅行季节：
桂林最适合旅游的季节是春季 4-5 月和秋季 10-11 月。

春季桂林雨水较多，山水烟雨朦胧，漓江两岸景色如画，非常适合摄影和山水观光。秋季天气相对凉爽，能见度较好，适合漓江游船、阳朔骑行、登山看日落和户外活动。

夏季 6-8 月多雨，但漓江水位较高，竹筏漂流和游船体验通常更好。不过夏季也较热，户外活动要注意防晒、防暑和防雨。冬季 1-2 月较冷，游客相对少，住宿和部分项目价格可能更低，适合想避开人群的游客。

城市概览：
桂林是中国最著名的山水旅游城市之一，以“桂林山水甲天下”闻名。这里有漓江、象鼻山、两江四湖、阳朔、遇龙河、兴坪古镇、十里画廊等经典景点，喀斯特峰林、江河、田园和古镇共同组成了极具辨识度的山水景观。

桂林适合喜欢自然风光、摄影、慢旅行、亲子游和户外体验的游客。第一次来桂林建议安排 2-3 天；如果想深度游览阳朔、兴坪、遇龙河、龙脊梯田等地，可以安排 4-5 天以上。

交通信息：
桂林两江国际机场位于市区西南方向，距离市区约 30 分钟车程。打车到市区通常约 60-80 元，具体价格取决于目的地和路况。也可以乘坐机场大巴前往市区。

桂林有多个火车站，其中桂林北站和桂林西站承担较多高铁线路，桂林站距离市区更近。桂林北站到市区可乘公交、出租车或网约车，通常约 30 分钟左右。

市内交通以公交、出租车和网约车为主。象鼻山、两江四湖、正阳步行街、东西巷等市区景点距离较近，适合步行或短途打车。

从桂林前往阳朔可以乘坐班车、高铁到阳朔站再换乘，也可以选择漓江游船从桂林顺流而下到阳朔。班车车程通常约 1.5 小时；漓江游船通常约 4-5 小时，是经典观光路线。

交通建议：
- 漓江游船旺季建议提前预订，尤其是节假日。
- 桂林到阳朔有多种交通方式，想看风景推荐漓江游船，想节省时间可坐班车或高铁换乘。
- 阳朔内部景点较分散，可以选择电动车、自行车、包车或打车。
- 遇龙河竹筏和漓江竹筏受天气、水位和景区政策影响，出发前建议确认开放情况。
- 雨天桂林山水有烟雨感，不一定影响观景，但户外活动需要防滑和带雨具。

住宿区域推荐：
1. 象山景区附近
适合第一次来桂林和短途停留的游客。这里靠近象鼻山、两江四湖、正阳步行街等景点，交通和餐饮方便。

2. 解放桥 / 两江四湖附近
适合喜欢夜景和城市散步的游客。晚上可以欣赏两江四湖灯光和桥景，步行体验较好。

3. 正阳步行街 / 东西巷
适合喜欢购物、小吃和市区便利的游客。周边餐饮、夜市和交通都比较方便。

4. 阳朔西街
适合背包客、年轻游客和喜欢热闹夜生活的人。周边住宿、餐厅、酒吧和旅行服务集中，但晚上可能比较吵。

5. 遇龙河 / 十里画廊周边
适合喜欢自然风光、田园骑行和安静度假的游客。这里比西街更适合慢旅行和亲子游。

6. 兴坪古镇 / 兴坪渔村
适合喜欢自然风光、摄影和安静氛围的游客。靠近二十元人民币背景取景地，适合看漓江山水和日出日落。

热门景点：
1. 漓江
类型：河流 / 山水风光 / 游船
推荐游玩时间：半天
亮点：喀斯特峰林、江水、山水画卷、九马画山、黄布倒影
建议：桂林到阳朔的漓江游船是经典路线，旺季建议提前预订。雨天有烟雨山水的意境。

2. 象鼻山
类型：地标 / 城市景观 / 自然风光
推荐游玩时间：1-1.5 小时
亮点：桂林城市地标、象鼻造型、漓江与桃花江交汇
建议：适合第一次来桂林打卡，可与两江四湖、东西巷安排在同一天。

3. 两江四湖
类型：夜景 / 游船 / 城市漫步
推荐游玩时间：1.5-2 小时
亮点：城市水系、夜景灯光、日月双塔、桥梁景观
建议：晚上体验更好，可选择步行或乘船夜游。

4. 阳朔西街
类型：老街 / 美食 / 夜生活
推荐游玩时间：1-2 小时
亮点：酒吧、餐厅、特产店、背包客氛围、夜生活
建议：晚上最热闹，适合逛街和用餐，但商业化程度较高。

5. 遇龙河
类型：竹筏漂流 / 田园风光 / 自然风光
推荐游玩时间：2-3 小时
亮点：竹筏漂流、田园风光、山水倒影、乡村景色
建议：遇龙河比漓江更安静，适合轻松漂流和拍照。竹筏项目可能受天气和水位影响。

6. 十里画廊
类型：骑行 / 田园风光 / 山水景观
推荐游玩时间：2-4 小时
亮点：田园风光、山峰、骑行路线、月亮山周边
建议：适合骑自行车或电动车游览，夏季注意防晒。

7. 月亮山
类型：山景 / 观景点 / 日落
推荐游玩时间：1.5-2 小时
亮点：天然拱形山洞、阳朔山景、日落视角
建议：登山需要体力，傍晚看日落较好，雨天需注意路滑。

8. 兴坪古镇
类型：古镇 / 江景 / 摄影
推荐游玩时间：半天
亮点：古镇街巷、漓江风景、二十元人民币背景
建议：适合喜欢摄影和安静山水的人，可以与漓江竹筏或兴坪渔村安排在一起。

9. 九马画山
类型：江景 / 山水风光 / 摄影
推荐游玩时间：30-60 分钟
亮点：漓江山峰、山壁纹理、传统传说
建议：通常在漓江游船或兴坪路线中顺路观看。

10. 黄布倒影
类型：山水风光 / 摄影
推荐游玩时间：30-60 分钟
亮点：二十元人民币背景、漓江倒影、经典桂林山水
建议：晴天和清晨倒影效果更好。

11. 银子岩
类型：喀斯特溶洞 / 自然风光
推荐游玩时间：1.5-2 小时
亮点：溶洞、钟乳石、灯光景观
建议：适合雨天或夏季避暑路线，可与阳朔行程结合。

12. 龙脊梯田
类型：梯田 / 民族村寨 / 摄影
推荐游玩时间：一天或住一晚
亮点：梯田、壮族瑶族村寨、日出日落、田园景观
建议：距离桂林市区较远，建议单独安排一天或住一晚。春季灌水和秋季稻谷成熟时景色最佳。

美食推荐：
1. 桂林米粉
桂林代表性美食，常见于早餐，配卤水、锅烧、酸豆角、花生、辣椒等，口味鲜香。
适合：早餐、本地美食体验

2. 阳朔啤酒鱼
阳朔特色菜，通常使用漓江鱼搭配啤酒和酱料烹制，适合正餐体验。
适合：本地菜体验、多人用餐

3. 荔浦芋扣肉
桂林传统菜，芋头软糯，扣肉香浓，适合多人用餐。
适合：本地菜体验

4. 田螺酿
广西特色菜，将肉馅塞入田螺壳中烹制，味道鲜香。
适合：喜欢尝鲜的游客

5. 桂花糕
带有桂花香气的传统糕点，适合作为小吃或伴手礼。
适合：甜品爱好者、伴手礼

6. 油茶
桂林及周边地区特色饮品，味道独特，适合想体验本地风味的游客。
适合：本地文化体验

7. 竹筒饭
具有山地和民族风味的食物，适合在周边村寨或景区体验。
适合：本地小吃爱好者

一日游路线：
主题：漓江 + 阳朔精华

上午：
从桂林出发，选择漓江游船或漓江竹筏路线，欣赏奇峰异石、江水和山水画卷。

午餐：
抵达阳朔后，品尝阳朔啤酒鱼、桂林米粉、荔浦芋扣肉等本地美食。

下午：
游览阳朔西街，购买特产，感受背包客和小镇氛围。时间充足可以前往十里画廊或月亮山。

晚上：
傍晚可登月亮山或在遇龙河周边看田园日落。晚上观看《印象刘三姐》演出，或在阳朔西街散步。

二日游路线：
主题：桂林市区 + 阳朔经典路线

第 1 天：
上午：象鼻山
下午：正阳步行街、东西巷、日月双塔
晚上：两江四湖夜游

第 2 天：
上午：漓江游船前往阳朔
下午：阳朔西街、遇龙河竹筏或十里画廊
晚上：《印象刘三姐》演出或西街夜游

三日游路线：
主题：第一次来桂林和阳朔经典路线

第 1 天：桂林市区
上午：象鼻山
下午：东西巷、正阳步行街
晚上：两江四湖夜游

第 2 天：漓江与阳朔
上午至下午：漓江游船或竹筏前往阳朔
晚上：阳朔西街、啤酒鱼

第 3 天：田园与溶洞
上午：遇龙河竹筏
下午：十里画廊、月亮山或银子岩
晚上：返回桂林或继续住阳朔

四日游路线：
主题：轻松山水旅行

第 1 天：
抵达桂林，入住象山景区、两江四湖或正阳步行街附近。晚上两江四湖夜游。

第 2 天：
漓江游船到阳朔，下午阳朔西街，晚上《印象刘三姐》。

第 3 天：
遇龙河竹筏、十里画廊、月亮山，体验阳朔田园风光。

第 4 天：
兴坪古镇、二十元人民币背景、黄布倒影。也可以选择龙脊梯田一日游。

购物与伴手礼：
1. 桂林米粉
桂林最具代表性的美食，适合现吃体验，也有部分包装产品可购买。

2. 桂花糕
带有桂花香气的传统点心，方便携带，适合作为伴手礼。

3. 三花酒
桂林传统名酒之一，适合送礼或收藏。

4. 桂林辣椒酱
桂林特色调味品，适合喜欢下厨和吃辣的人。

5. 漓江画
以桂林山水为主题的画作或装饰品，适合作为纪念品。

6. 壮锦
广西民族特色织锦工艺品，适合作为文化类纪念品。

7. 罗汉果
广西常见特产，可泡水饮用，适合带回家。

旅行贴士：
- 漓江游船和热门竹筏路线旺季需提前预订。
- 遇龙河竹筏、漓江竹筏和游船可能受天气、水位影响，出发前建议确认开放情况。
- 阳朔热门区域消费相对较高，购物和用餐前可以先看价格。
- 桂林米粉通常早上吃最有本地氛围，推荐选择本地人多的小店。
- 雨天游漓江别有烟雨山水的意境，不必因为小雨取消行程。
- 夏季桂林潮湿炎热，户外游览要注意防晒、防蚊和补水。
- 阳朔骑行或骑电动车时注意交通安全，乡间道路夜间照明有限。
- 两江四湖夜游适合安排在桂林市区住宿的晚上。
- 如果想看更自然的山水风光，可以在阳朔或兴坪住一晚。
- 龙脊梯田距离较远，建议单独预留一整天或住一晚。
